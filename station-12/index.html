Crew & Settings

How fairness works: Each shift, the rotation picks whoever's present with the lowest count. Ties break by newest-first. Detailed-off members are skipped automatically (their count doesn't move, so they catch up naturally on return). Lt. Martin and Sgt. Glover are marked exempt and stay out of the auto-pick.

Backup Watch: Picked from the opposite rig of the primary watch holder — if primary is Engine 12, backup comes from Rescue 12 (and vice versa). Each crew member's rig is set with the red/blue button in the row above (tap to cycle Engine → Rescue → none). Backup watch tracks its own count.
