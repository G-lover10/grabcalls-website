Behavior

How fairness works: Each shift, the rotation picks whoever's present with the lowest count. Detailed-off members are skipped automatically (count doesn't move → catches up naturally on return). Lt. Martin and Sgt. Glover are exempt.

Tiebreaker chain (when two crew are tied at lowest count):

  1. Detail break: if one of the tied crew was detailed off last shift, they lose the tie (give them a breather).
  2. Medic break (watch/backup only): medics on Rescue lose ties — they're acting officer, so the non-medic takes it.
  3. Newest first: if still tied, newest member takes it (Smart → Wilson → … → Andrews).

Backup Watch: Always from the opposite rig of primary. Uses today's lineup, so if you put Bentley on Engine for the night, the algorithm respects that.

Meals: Designated cook (Holt) auto-picks when present. When Holt's off, fair rotation among others.
